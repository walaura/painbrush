## Wuth

Imagine you wanna make open graph images but spinning up a browser feels wasteful. Now imagine you could make something even more wasteful. This is one of the may uses of Painbrush, which lets you create bitmap images (yeah, those) using the javascript you are familiar with.

Not canvas or anything, that sounds hard, brand new api.

Ahh!! Because this is really meant for og images on my blog fonts are a big deal. You are missing out if you dont make a little pixel font for your usage, or at least package `lucas`.

This is a monorepo bc im fancy like that, so:

- You can read an intro to painbrush on [packages/painbrush](/packages/painbrush/)
- You can see an example of many features of painbrush in use at [packages/example](/packages/example/)
- You can painbrush running on your browser at [packages/example-browser](https://walaura.github.io/painbrush/). its awful. use canvas.
